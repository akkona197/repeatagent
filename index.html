Nhập Data

Paste data từ Google Sheets vào đây · Dashboard tự động tính toán khi bấm Parse & Apply
Chưa có data. Paste và Parse để bắt đầu.
Format: Tab-separated (copy thẳng từ Google Sheets). Cần có header row.
Kéo trực tiếp vùng A6:M sheet "3. Agent list Weekly" (cần deploy Apps Script — xem hướng dẫn). URL chỉ lưu trên trình duyệt này.
Format: Tab-separated. Bao gồm header row có ngày (7/13/2026, 7/6/2026...) để xác định tuần.
⚠
Lưu ý quan trọng: Mỗi khi muốn nhập data mới cho tuần tiếp theo, cần F5 (reload trang) trước để reset toàn bộ trạng thái. Dashboard không lưu data giữa các phiên.
Hướng dẫn sử dụng — Dành cho người mới
Quickstart — Chạy thử ngay
Nếu chưa có data thật, bấm Load Sample Data (W28) ở trên → rồi bấm Parse & Apply → Dashboard sẽ render ngay với data mẫu W28.
Quy trình nhập data thực mỗi tuần
Bước 1 — File 1: Agent List Weekly
  1. Mở Google Sheets → sheet Agent List Weekly
  2. Click vào ô A1 (cột Week)
  3. Bôi đen toàn bộ vùng data từ A1 đến M cuối
    (Ctrl+Shift+End để chọn đến ô cuối cùng)
  4. Ctrl+C để copy
  5. Paste vào ô bên trái trong trang này
Script chỉ lấy dòng có cột Repeated = số (0, 1, 2, 3).
Dòng có Repeated = "-" sẽ tự bỏ qua.
Nên paste nhiều tuần để có Trend chart.
Bước 2 — File 2: CSAT Weekly
  1. Cùng file Sheets → sheet CSAT_Weekly
  2. Click vào ô C1 (cột Agent name)
  3. Bôi đen từ C1 đến S cuối
    (Bao gồm cả dòng header ngày tháng và dòng W29/W28...)
  4. Ctrl+C để copy
  5. Paste vào ô bên phải trong trang này
Script tự detect dòng chứa ngày (7/13/2026...) để xác định tuần.
CSAT map theo email → cần email khớp với File 1.
File 2 có thể paste không cần File 1 (dùng riêng cho tab History).
Bước 3 — Parse & xem kết quả
Parse & Apply
Bấm nút xanh để parse. Status bar sẽ hiển thị số rows đã đọc được. Nếu lỗi sẽ thấy thông báo đỏ kèm nguyên nhân.
Dashboard
Sau khi parse thành công, chuyển sang tab Dashboard để xem KPI, chart, và bảng agent detail. Có thể nhập Root Cause trực tiếp vào bảng.
Report
Tab Report auto-generate text theo format chuẩn IH. Có nút Copy từng section hoặc Copy toàn bộ để paste vào chat/email.
Tuần mới: F5 trang trước khi paste data mới. Data chỉ tồn tại trong phiên làm việc hiện tại, đóng tab là mất.
